# Security Policy

Talkto Reliable is currently a private, proprietary package.

No public vulnerability disclosure address has been approved or committed yet. Before any public release, maintainers must define a private disclosure channel, response owner, supported versions policy, and public security contact.

Until then, report suspected vulnerabilities through the approved internal project channel. Do not include real shared secrets, production payloads, or private credentials in issue text, pull requests, logs, screenshots, or documentation.
