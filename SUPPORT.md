# Support

Laravel Talkto is open-sourced software licensed under the MIT license.

Use the repository issue process or the maintainer-approved public support channel for questions, bug reports, and release questions.

When reporting an issue, include:

- package version or Git tag
- Laravel version and PHP version
- database and queue driver, if relevant
- exact command output for failed commands
- sanitized `talkto:trace` output for affected messages, when applicable
- sanitized `talkto:security-audit` output when configuration or signing is involved
- a minimal generic reproduction with placeholder service names such as `website-service` and `inventory-service`

Do not publish secrets, credentials, production payloads, raw signatures, raw nonce values, private URLs, cookies, authorization headers, or private service names.

Security-sensitive reports should follow [SECURITY.md](SECURITY.md).
